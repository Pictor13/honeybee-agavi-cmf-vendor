## Available Types

Available Dat0r types are:

- Aggregate
- Boolean
- Email
- IntegerCollection
- Integer
- KeyValue
- KeyValuesCollection
- Reference
- Select
- Textarea
- TextCollection
- Text
- Uuid

Suggestions for other types include:

- Phone
- FileCollection
- File
    - Audio
    - Binary (OctetStream)
    - Document
    - Image
    - File
    - Video
- specific Aggregates like
    - Paragraph
    - GalleryItem
    - Gallery
    - ParagraphWithImage
    - etc.
