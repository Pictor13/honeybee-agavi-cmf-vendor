The default layout is defined by the ```app/templates/Master.twig``` template
in addition to the view templates and the slots embedded in them.

## Rough Structure

```
Page

    pageHeader

        applicationLogo

        globalSearch

        userWidget

    pageSection

        pageInfo

            mainNav

            tocTitle

            notifications

        pageMain

            mainContent

                toolbar

                content

                actionbar

            sidebars

                sidebarsNav

                sidebarsContent

    pageFooter
```
